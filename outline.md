# 📁 Project: 업무 말투 변환기 (Business Tone Converter)

## 1. 🎯 프로젝트 소개 (Introduction)
**"하고 싶은 말은 있는데, 어떻게 격식 있게 표현해야 할지 모르겠다면?"**

본 프로젝트는 사용자가 전달하고자 하는 **핵심 내용(Core Message)**만 입력하면, 수신자(상사, 타팀 동료, 고객 등)의 관계에 맞춰 적절한 **비즈니스 매너가 갖춰진 업무용 말투**로 자동 변환해 주는 웹 애플리케이션입니다. 업무 커뮤니케이션에 어려움을 겪는 직장인들의 고민을 해결하고 작성 시간을 단축하는 것을 목표로 합니다.

## 2. ✨ 핵심 기능 (Key Features)

### A. 핵심 내용 입력
- 사용자가 변환하고 싶은 문장의 핵심 키워드나 날것의 문장을 입력합니다.
- *예시: "이거 안 됨. 다시 해줘."*

### B. 수신자 대상 선택 (Persona Selection)
- 메시지를 받는 대상에 따라 톤앤매너(Tone & Manner)를 조정합니다.
    1. **상사 (Boss):** 정중하고 보고 형식에 맞는 어조
    2. **타팀 동료 (Colleague):** 협조적이고 정중하지만 너무 딱딱하지 않은 어조
    3. **고객 (Client):** 서비스 지향적이고 친절하며 격식 있는 어조

### C. 실시간 변환 및 결과 출력
- 선택한 옵션에 맞춰 변환된 텍스트를 제공합니다.
- *예시 (상사 선택 시): "현재 해당 사항에 진행이 어려운 부분이 있어, 재검토를 부탁드려도 될지 여쭙습니다."*

## 3. 🛠 기술 스택 (Tech Stack)

이 프로젝트는 빠르고 효율적인 배포와 안정적인 백엔드 처리를 위해 아래와 같은 스택을 사용합니다.

| 구분 | 기술 (Technology) | 설명 |
| :--- | :--- | :--- |
| **Frontend** | **HTML5, CSS3** | 가볍고 직관적인 UI 구성을 위한 정적 웹 기술 |
| **Backend** | **Python (Flask)** | 텍스트 변환 로직 처리 및 API 서버 구축 |
| **Infrastructure** | **Firebase** | 호스팅(Hosting) 및 배포 관리 |

## 4. 🔄 시스템 아키텍처 및 워크플로우 (User Flow)

1. **접속:** 사용자가 웹 페이지에 접속 (Firebase Hosting)
2. **입력:** 텍스트 입력 및 수신자 대상 선택 후 '변환' 버튼 클릭
3. **요청:** Frontend에서 Python Flask 서버로 API 요청 (POST)
4. **처리:** Flask 서버에서 입력된 텍스트를 대상에 맞게 변환 (로직/AI 처리)
5. **응답:** 변환된 텍스트를 Frontend로 반환하여 화면에 표시
